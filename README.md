# KertFlight

**Requirements**
- Docker

**How to run the project**
1. ``` cd path/to/project/KertFlight/ ```
2. ``` docker build -t kertflight . ```
3. ``` docker run --rm -it -p 8888:8080 kertflight ```
4. Visit http://localhost:8888 to see if project is running!
